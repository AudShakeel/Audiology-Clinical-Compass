# The Audiology Clinician Resource Hub

Created by Tahreem Shakeel, AuD Candidate, University of the Pacific, San Francisco, CA.

Open `index.html` to view the site. The site is static and can be hosted through GitHub Pages, Netlify, Vercel, Cloudflare Pages, or a university server.

Included:
- Full static website
- 42 original printable handouts
- Hearing aid troubleshooting center
- Expanded outcome-measure page across subspecialties with 101 entries
- Genetics and syndrome handouts
- Research summaries for counseling and clinical decisions
- Evidence source tracker
- Permission status tracker
- Reviewer brief
- Deployment guide

Clinical and legal safeguards:
- Original wording only for hosted handouts
- Official links only for restricted tools and commercial materials
- No manufacturer software screenshots
- No proprietary test forms, audio, sentence lists, manuals, or textbook figures
- Last-reviewed date on pages
