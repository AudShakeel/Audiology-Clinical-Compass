GitHub Upload Guide

Use the existing repository. Do not create a new one.

  1. Download this final ZIP.
  2. Right-click the ZIP and select Extract All.
  3. Open the extracted folder.
  4. Select everything inside the folder with Ctrl + A.
  5. Go to the GitHub repository.
  6. Click Add file, then Upload files.
  7. Drag all selected files and folders into GitHub.
  8. Use commit message: Final Audiology Clinical Compass integrated launch update.
  9. Leave Commit directly to the main branch selected.
  10. Click Commit changes.
  11. Wait 1 to 3 minutes for GitHub Pages to update.
  12. Refresh the live site with Ctrl + F5 if the old version remains cached.

GitHub Pages check

Settings > Pages should show: Source = Deploy from a branch, Branch = main, Folder = /(root).
